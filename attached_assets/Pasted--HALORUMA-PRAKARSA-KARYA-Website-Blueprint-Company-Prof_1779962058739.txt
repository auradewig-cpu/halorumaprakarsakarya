### Typography System

```

Weight: 300 (Light) for large display, 600 (SemiBold) for emphasis
Use: Hero titles, section headers, quotes

SUBHEADING / NAVIGATION
Font: "Syne" — Geometric, modern, architectural
Weight: 400 Regular / 700 Bold
Use: Nav links, subheadings, labels

BODY / PARAGRAPH
Font: "Lato" — Clean, humanist, readable
Weight: 300 Light / 400 Regular
Use: Paragraphs, descriptions, captions

ACCENT / DETAIL
Font: "Playfair Display" — Classical elegant serif
Weight: 400 Italic
Use: Pull quotes, testimonials, section accents

MONOSPACE / DETAIL
Font: "DM Mono"
Use: Statistics numbers, counters, coordinates
```

## ⚡ INTERACTION & ANIMATION SPECS

```
PAGE LOAD:
- Logo draws itself (SVG path animation, 1.2s)
- Nav items fade in staggered (0.1s each)
- Hero text lines slide up (0.15s delay each line)
- Hero image reveals with clip-path wipe (0.8s ease)

SCROLL ANIMATIONS:
- Elements: fade-up + slight translateY (30px → 0)
- Trigger: 80% into viewport
- Duration: 0.6s ease-out
- Stats: count-up animation on enter

HOVER STATES:
- Cards: translateY(-6px) + shadow deepens
- Images: scale(1.04) with overflow hidden
- Buttons: background slides in from left
- Nav links: underline grows from center

CURSOR: Custom cursor — small circle + architectural crosshair
(desktop only)

TRANSITIONS:
- Page transitions: horizontal slide with copper gold flash bar
- Image loading: blur-to-sharp (blur placeholder)

PORTFOLIO FILTER:
- Masonry grid reflow with smooth animation (GSAP Flip)

PARALLAX:
- Hero image: 20% scroll offset
- Section backgrounds: 10% offset
```

## 🎨 MOOD BOARD DIRECTION

```
AESTHETIC DIRECTION: "Warm Brutalist Luxury"
─────────────────────────────────────────────
Combine the raw confidence of brutalism with 
the warmth of Japandi/wabi-sabi and the 
precision of Swiss design — creating a brand 
that feels simultaneously bold and inviting.

REFERENCES:
- Typography feel: Vogue × Wallpaper Magazine
- Spatial feel: Tokyo luxury hotel × Surabaya creative studio
- Color feel: Natural stone, aged copper, warm concrete

DESIGN PRINCIPLES:
1. Every element earns its place — no decoration for decoration's sake
2. Photography is the hero — design serves the images
3. Generous white space = confidence
4. Copper gold details = signature luxury touch
5. Architectural grid system throughout

AVOID:
✗ Purple/blue gradient tech-startup aesthetics
✗ Clip art or generic stock illustrations
✗ Busy layouts with too many elements competing
✗ Comic Sans or decorative script fonts
✗ Generic construction iconography (hard hats, etc.)
```
